RGB vs CMYK

diferença entre os modelos de cores

O que é RGB?

RGB (Red, Green, Blue) é um modelo de cores usado em telas. A mistura dessas três cores em diferentes intensidades cria todas as cores visíveis.

Como funciona um pixel RGB?

Em telas, cada pixel é formado por três subpixels: vermelho, verde e azul. Quando combinados em diferentes cores, criam as cores que vemos na tela do computador por exemplo.

O que é CMYK?

CMYK (Cyan, Magenta, Yellow, Black) é um modelo de cores, usado na impressão. Ele mistura tintas para formar as cores desejadas no papel.

Comparação RGB x CMYK

RGB é melhor para telas, enquanto CMYK é essencial para impressão. RGB usa luz, enquanto CMYK usa tinta, o que causa diferenças na percepção de cada cor.
